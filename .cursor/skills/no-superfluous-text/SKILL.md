# No Superfluous Text

## Instructions

- Do not add explanatory UI text unless it is needed for action, orientation, or accessibility.
- Prefer the visual structure and existing controls over helper paragraphs.
- Remove titles, descriptions, captions, or status copy that repeat what the UI already shows.
- Keep necessary labels short and functional.
- When uncertain, omit decorative or explanatory copy and ask only if the UI becomes unclear.
